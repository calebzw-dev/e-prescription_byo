# E-Prescription System Bulawayo
Final year BSc Computer Science project, NUST 2024.

Web app to help allocate medical attention faster by digitizing prescription 
records across different pharmacies in Bulawayo.

**Tech Stack:** [Replace with yours: PHP, MySQL, HTML, CSS, JavaScript]
**Core Features:**
- Patient registration and login
- Doctor prescription entry
- Pharmacy dashboard to view/fulfill prescriptions
- Admin panel for record management

**Future Scope:** AI chatbot integration for basic home-care guidance.

**Status:** Core web app complete. Built and tested locally during degree.
